# Approvazioni: il widget Approvazioni personali visualizza le bozze che non si trovano nella fase a cui è assegnato l’utente

Quando un utente visualizza il widget Le mie approvazioni in Home, visualizza le bozze in cui l’utente è assegnato a una fase, ma la bozza non si trova nella fase a cui l’utente è assegnato. Questo può rendere difficile per l’utente vedere rapidamente quale azione è necessaria.

_Segnalato per la prima volta il martedì 16 febbraio 2026._
