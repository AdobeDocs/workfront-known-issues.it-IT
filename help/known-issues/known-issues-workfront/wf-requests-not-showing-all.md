# Richieste: la vista &quot;All&quot; (Tutti) non mostra tutte le richieste

>[!NOTE]
>
>* Questo problema è stato segnalato nella nuova esperienza di richiesta.
>* Questo problema è stato risolto il giovedì 18 febbraio 2026.

Quando un utente visualizza l’elenco delle richieste nella nuova esperienza di richiesta e la visualizzazione è impostata per mostrare tutte le richieste, vengono visualizzate solo alcune delle richieste.

_Segnalato per la prima volta il mercoledì 16 dicembre 2025._
