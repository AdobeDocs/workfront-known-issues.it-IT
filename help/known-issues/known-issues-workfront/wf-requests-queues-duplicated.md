# Richieste: le code di richieste vengono visualizzate più volte

>[!NOTE]
>
>* Questo problema è stato risolto il venerdì 8 gennaio 2026.
>* Questo problema è stato segnalato nell’esperienza di reporting legacy.

Quando un utente inizia a creare una richiesta e seleziona una coda di richieste, la stessa coda di richieste viene visualizzata più volte nell’elenco a discesa.

_Segnalato per la prima volta il mercoledì 30 dicembre 2025._
