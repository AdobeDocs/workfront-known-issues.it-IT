# Richieste: le code delle richieste vengono visualizzate più volte

>[!NOTE]
>
>* Questo problema è stato risolto l’8 gennaio 2026.
>* Questo problema è stato segnalato nell’esperienza di reporting precedente.

Quando un utente inizia a creare una richiesta e seleziona una coda di richieste, quest’ultima viene visualizzata più volte nell’elenco a discesa.

_Segnalato per la prima volta il 30 dicembre 2025._
