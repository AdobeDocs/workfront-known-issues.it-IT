# Bozze: impossibile generare una bozza da un file PowerPoint

Quando un utente tenta di generare una bozza da un file PowerPoint, l’operazione non riesce e viene visualizzato un messaggio di errore.

**Soluzione alternativa**

Esporta il file PowerPoint in PDF, quindi crea una bozza dal PDF.

_Segnalato per la prima volta il 21 marzo 2025._
