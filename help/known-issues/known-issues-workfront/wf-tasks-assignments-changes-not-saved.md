# Attività: le modifiche apportate alla sezione Assegnazioni di un’attività non vengono salvate

>[!NOTE]
>
>Questo problema si applica all’esperienza attività legacy.

Quando un utente apporta modifiche alla sezione Modifiche di un’attività nella finestra di dialogo delle attività legacy e le salva, le modifiche non vengono salvate.

**Soluzione alternativa**

Per alcune modifiche, puoi utilizzare la nuova esperienza di attività.

_Segnalato per la prima volta il martedì 24 novembre 2025._
