# Utenti: gli utenti rimossi da Adobe Admin Console sono ancora attivi in Workfront

Quando un amministratore rimuove un utente nel Adobe Admin Console, non viene rimosso dal Workfront. L’utente è ancora visibile e attivo in Workfront.

_Segnalato per la prima volta il giovedì 3 febbraio 2026._
