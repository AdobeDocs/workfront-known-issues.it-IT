# Moduli personalizzati: impossibile aggiungere o rimuovere in blocco moduli personalizzati nelle attività dei modelli

Se un utente tenta di aggiungere o rimuovere in blocco un modulo personalizzato in un’attività modello, il modulo non viene aggiunto o rimosso e l’utente visualizza il seguente errore:

[!UICONTROL Riproviamo. Parametro non valido: valore templateID &quot;XXXXXXXXXXXXXX&quot;]

Se l&#39;utente individua il modello con il GUID specificato e tenta di aggiungere o rimuovere moduli personalizzati nelle altre attività del modello, l&#39;errore si verificherà nuovamente utilizzando un altro ID modello.

I moduli personalizzati possono essere aggiunti o rimossi con un’unica attività modello. Questo errore si applica solo all&#39;aggiunta o alla rimozione in blocco.

_Segnalato per la prima volta il 10 novembre 2023._
