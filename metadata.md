# Metadati per uso interno

Il file [!DNL metadata].md include metadati a livello di repository che passano attraverso i file [!DNL TOC.md] della guida utente nel repository. Se desideri modificare il contenuto del file metadata.md per ogni guida utente, effettua questa operazione in tutti i file TOC.md.

Vedi [Metadati](https://experienceleague.adobe.com/docs/authoring-guide-exl/using/editing/user-guide-setup/metadata.html?lang=it) per un elenco di valori validi per **soluzione**, **prodotto** e **tipo**.
