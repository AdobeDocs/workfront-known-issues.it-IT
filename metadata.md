# Metadati per uso interno

Il file [!DNL metadata].md include metadati a livello di repository che passano ai file della guida utente [!DNL TOC.md] nel repository. Se desideri modificare il contenuto del file metadata.md per qualsiasi Guida utente, effettua questa operazione in qualsiasi file TOC.md.

Per un elenco di valori validi per **soluzione**, **prodotto** e **tipo**, consulta [Metadati](https://experienceleague.adobe.com/docs/authoring-guide-exl/using/editing/user-guide-setup/metadata.html).
